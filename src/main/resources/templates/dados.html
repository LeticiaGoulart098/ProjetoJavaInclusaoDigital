Banco de Dados

Conceitos, modelos e aplicações essenciais

O que é um Banco de Dados?

Um banco de dados é um sistema organizado para armazenar e gerenciar grandes volumes de informações de forma estruturada, permitindo acesso rápido, seguro e eficiente.

Componentes de um Banco de Dados

Dados: Fatos brutos como nomes, números, datas e textos.

SGBD: Software responsável por gerenciar o banco. Exemplos: MySQL, PostgreSQL, Oracle.

Usuários: Pessoas e sistemas que utilizam o banco.

Modelos de Dados

Modelo Relacional

Organiza dados em tabelas relacionadas por chaves.

Modelo Hierárquico

Estrutura em formato de árvore.

Modelo de Rede

Permite múltiplas relações entre registros.

Orientado a Objetos

Armazena dados no formato de objetos.

Funcionalidades dos Bancos de Dados

Principais operações: consulta, inserção, atualização e remoção.

Integridade: Mantém dados consistentes.

Segurança: Protege contra acessos indevidos.

Backup: Evita perda de informações.

Tipos de Bancos de Dados

SQL

Estruturados e relacionais.

NoSQL

Flexíveis e ideais para grandes volumes.

In-Memory

Armazenados na memória para alta performance.
