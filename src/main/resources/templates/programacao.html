Principais Linguagens de Programação

Conceitos Fundamentais

Linguagens de Programação

Existem muitas linguagens de programação, cada uma com suas próprias sintaxes e usos.

Algoritmos

Um algoritmo é uma sequência de passos que resolvem um problema específico.

Estruturas de Controle

As estruturas de controle determinam o fluxo de execução de um programa:

  • Sequência: execução linha por linha;
  • Decisão: condições (if / else);
  • Repetição: loops (for / while).

Variáveis e Tipos de Dados

Variáveis armazenam dados que podem ser manipulados no programa.

Funções

Funções são blocos de código reutilizáveis que executam tarefas específicas.

Exemplos Práticos

Exemplo em Python:

# Este é um comentário
nome = "Mundo"
print("Olá, " + nome + "!")

Estrutura de Decisão:

idade = 18
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

Loop:

for i in range(5):
    print("Este é o número", i)
