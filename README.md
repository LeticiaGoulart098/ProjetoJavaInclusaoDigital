# ProjetoJavaInclusaoDigital

Plataforma web de cursos gratuitos em tecnologia, desenvolvida com Java Spring Boot e integrada ao banco de dados MySQL.

O projeto tem como objetivo promover inclusão digital, organizando conteúdos gratuitos de programação e tecnologia para estudantes iniciantes.

## Tecnologias utilizadas

- Java
- Spring Boot
- MySQL
- HTML5
- CSS3
- JavaScript
- Thymeleaf

## Funcionalidades

- Plataforma de cursos gratuitos de programação
- Organização de conteúdos em módulos
- Interface responsiva em tema dark
- Sistema preparado para cadastro e banco de dados
- Formulário de contato integrado ao banco MySQL
